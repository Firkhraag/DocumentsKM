Клиент лежит в папке client, написан на React. Сборка "npm run build" использует webpack и помещает готовый бандл в ClientApp/dist в папке сервера.
Для development используется "npm run watch", чтобы следить за изменениями.

При изменении шрифтов в таблицах шаблонов word нужно менять отступ верхнего поля, чтобы таблицы соединялись с headerом.
Word может добавить лишний footer при изменении файла, поэтому приходится менять индекс FooterParts в OpenXML.

Файл миграций: data/migration_sql_server.ipynb представляет собой документ Jupyter Notebook. Для запуска используется Python 3.
Там же лежат sql файлы таблиц.

Тесты лежат в папке server отдельно от сервера. Для запуска: "dotnet test".
